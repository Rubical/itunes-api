## How to Run the App

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/Rubical/test-2.git
   ```
2. Navigate to the project directory:
   ```bash
   cd test-2
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Run the application:
   ```bash
   npm run dev
   ```
